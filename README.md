# Jack's Apartment
## A Short Video Game by Miah Terrazas & Zion Hsieh
### Original known as Project Noneuclidean Horror Game

## Setup
*You find yourself in the home of a man named Jack, a person shrouded in mystery. Can you manage to escape from his apartment, or will you find out why he's the recluse he is?*
**Made with Unity.**

## Features
- Player Interactability with the World via inspecting game objects
- A Player Inventory
- A patrolling, pursuing Enemy, named Jack
- Multiple different Puzzles
- A Note system for the player to understand the world
- Noneucldian doors that make no sense in space
